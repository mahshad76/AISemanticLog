# Project Overview ⚡️
This project is a Jetpack Compose application that displays a searchable list of log entries. Users can search for specific logs and view detailed information about each entry on a separate screen.
The logs are grouped by date, with each group displayed under a sticky header. A severity indicator in each header visualizes the distribution of log severity levels within the corresponding group of filtered results.
The app is built with a modern Android tech stack, leveraging dependency injection, a modular architecture, and network communication for data fetching.
